# Инструкция по сборке

## Требования
- CMake 3.20+
- Компилятор с поддержкой C++20 (MSVC 2019+/MinGW Clang/LLVM)
- Qt 5.15+ или Qt 6 (для графического интерфейса). Если Qt не найден, проект соберётся в консольном режиме.

## Шаги
1. Создайте папку сборки:
   ```
   cmake -S . -B build -DFS_ENABLE_QT=ON
   ```
2. Соберите бинарник:
   ```
   cmake --build build --config Release
   ```
3. Исполняемый файл `forensic_scanner` появится в каталоге `build` (или `build/Release` под Windows).

## Настройка
- Основные пути и списки задаются в `config/config.json`.
- База хешей и триггеров по умолчанию берётся из `hashes.txt` и `triggers.txt` в корне проекта.
