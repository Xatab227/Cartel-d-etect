# Форензик-/античит-сканер (C++)

Пример десктопного сканера для Windows x64, реализованный на стандартном C++17.

## Возможности
- Анализ хешей файлов в системных папках и пользовательских директориях.
- Поиск ключевых слов и доменов в экспортированной истории браузера.
- Проверка логов Discord на ключевые слова и названия серверов.
- Подсчёт итогового процента риска с цветовой индикацией.
- Консольный вывод с блоками логов и прогресс-баром.

## Структура проекта
```
CMakeLists.txt
config/
  config.ini
  hashes.csv
  keywords.txt
  sites.txt
  sample_browser_log.txt
  sample_discord_log.txt
src/
  main.cpp
  Config.*
  Logger.*
  HashScanner.*
  BrowserScanner.*
  DiscordScanner.*
  RiskCalculator.*
  ui/MainWindow.*
  utils/SHA256.*
```

## Сборка (Windows + CMake)
1. Установите [CMake](https://cmake.org/) и компилятор с поддержкой C++17 (MSVC/MinGW).
2. В корне проекта выполните команды:
   ```bash
   cmake -S . -B build
   cmake --build build
   ```
3. Запустите бинарный файл `build/ForensicScanner` в директории проекта. Для демонстрации логов по умолчанию используются примеры из папки `config/`.

## Конфигурация
- Файл `config/config.ini` задаёт пути к базам и логам, а также веса риска.
- Списки ключевых слов (`keywords.txt`), сайтов (`sites.txt`) и серверов Discord (поле `discordServers` в `config.ini`) можно расширять без перекомпиляции.
- База хешей ожидается в формате `категория,хеш`.

## Ограничения демо-версии
- UI реализован как консольный прототип. Для полноценного графического интерфейса можно заменить `ui/MainWindow` на Qt/WinAPI-реализацию, сохранив интерфейсы модулей.
- Сканирование использует стандартную библиотеку и не модифицирует файлы пользователя.
